## Getting Started


### Prerequisites
- web browser (e.g. Google Chrome, Firefox)
- text editor (e.g. VSCode)
- CLI/terminal

### Clone this repository

```
git clone git clone https://github.com/Efiamotu-1/To-do-list-review.git
```
### Open the project in the browser

```
npm run start
```
